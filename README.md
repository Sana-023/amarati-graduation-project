Amarati is a smart residential building management system designed to improve communication, maintenance management, and digital services for residential communities.

The system connects owners, tenants, supervisors, and maintenance providers through a unified digital platform.

## Main Features
- User management with role-based access (Owner, Tenant, Supervisor, Provider)
- Personal and community maintenance requests
- Verified maintenance provider selection
- Digital payments and billing
- Community voting for shared maintenance
- Document management (Digital Unit Passport)
- Notifications and reminders
- AI chatbot assistant
- Predictive maintenance module

## Project Status
This project is currently under development as a graduation project.
The repository contains system requirements and design artifacts.
